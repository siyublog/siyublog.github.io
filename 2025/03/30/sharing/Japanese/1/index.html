文法（ぶんぽう）

1.肯定句：名は　名です。

は：主格助词，无意义，读作wa

です：是

2.否定句：名は　名では　ありません。（名は　名じゃない。）

例：

他不是美国人：彼（かれ）はアメリカ人ではありません。

她：彼女（かのじょ）

3.疑问句：名は　名ですか。

句尾的か：升调+浊化

例：

あなたは　小野さんですか。

ーはい、小野です。

ーはい、そうです。

ーいいえ、そうではありません（ちがいます）。

4.属性：名の名

句尾加です

日常用语

谢谢：ありがとう　ございます（尊敬语）

询问年龄（尊敬）：おいくつ　ですか

询问年龄：何（なん）歳（さい）ですか

文法

1.事物代词：これ／それ／あれは　名です。

由近及远：これ（这个）／それ（那个）／あれ（那个）

どれ（哪一个）

例：

これ　は　雑誌　です。

それ　は　カメラ　では　ありません。

あれ　は　パソコン　ですか。

山下（やました）さん　の　かぼん　は　どれ　ですか。

2.连体词：この／その／あの+名は　名です。

例：

この　ペン　は　わたし　の　です。

その　いす　は　わたし　の　じゃない。

あの　新聞　は　先生　の　ですか。

王さん　の　カメラ　は　どの　カメラ　ですか。

3.所属及其他：名　の　名　です。（所属、内容、产地）

所属：わたし　の　名前（なまえ）。

内容：日本語　の　雑誌（ざっし）。

产地：日本　の　製品（せいひん）。

その　英語（えいご）　の　辞書　は　王さん　の　です。

4.人称疑问代词（谁）：だれ

普通（ふつう）：だれですか。

礼儀（れいぎ）：どなたですか。

尊敬（そんけい）：どちら様（さま）ですか。

例：

この　手紙（てがみ，信的意思）　は　だれ　の　ですか。

その　机（つくえ）　は　どなた　の　ですか。

5.疑问代词（什么）：何（なん）

注意与 哪一个 区别：どれ（不接名词）／どの（接名词）。

これ　は　何　ですか。

それ　は　何　の　雑誌　ですか。

日本語　の　辞書　は　どれ　ですか。

表达及词语

1、尊敬：かた（方）

例：あの方は田中さんです。

2、询问年龄

何歳ですか。＝　何歳？

おいくつですか。＝　いくつ？

3、请：どうぞ

用以给对方物品，或劝对方进餐。

例：これは　中国の名産品です。どうぞ。

ー　どうも　ありがとうございます。

基本课文

1、

甲：これ　は　テレビですか。

乙：いいえ、それは　テレビ　ではありません。パソコンです。

甲：それは　何ですか。

乙：これは　日本語の本です。

2、

甲：森さんのかばんは　どれですか。

乙：あのかばんです。

甲：そのノート　は　だれのですか。

乙：私のです。

应用课文

1、

李さん、それは何ですか。

これですか。家族の写真です。

この方はどなたんです。

わたしの母です。

お母さんは　おいくつですか。

五十二歳です。

2、

小野さん、これ、どうぞ。

えっ、何ですか。

お土産です。

わあ、ミルクの　ハンカチですか。

ええ。スワトウの　ハンカチです。中国の名産品です。

どうも　ありがとう　ございます。
